# キャンパスカーシェア予約アプリ（デモ版）

大学のキャンパス間移動を想定した、**カーシェア予約アプリのデモ版**です。  
このリポジトリはブラウザのみで動作する簡易デモで、外部ライブラリを使用していません。

---

## 🔑 ログイン情報（デモ用）
- 学籍番号：`c000000`  
- パスワード：`000000`

（ログイン情報は [`DUMMY_USERS`](script.js) に定義されています）

---

## 🧭 主な機能（現在の実装に基づく）
- 学籍番号とパスワードによるデモログイン（クライアントサイドのみ）。
- 複数キャンパスの一覧表示と選択（データは [`CAMPUSES`](script.js)）。
- 日付・開始時刻・終了時刻の選択（15分刻み）。時刻選択ロジックは [`populateTimeSelects`](script.js) で生成。
- 予約は同日同キャンパスで重複しないようチェック（重複判定は [`isTimeAvailable`](script.js)）。
- 予約完了時、終了時刻のあと30分が自動でブロックされます（定数：`RESERVATION_BUFFER_MINUTES`）。
- 予約データはブラウザの `localStorage` に保存／読み込み（`loadReservations` / `saveReservations`）。
- 現状の挙動：ログイン時に `localStorage` の予約を一度クリアする実装あり（デモ用途）。

---

## ⚙️ 実行方法
ローカル環境で動かすには、ブラウザで `index.html` を開くだけです（例: VS Code の Live Server などを推奨）。

- 開くファイル: [index.html](index.html)
- スクリプト: [script.js](script.js)
- スタイル: [style.css](style.css)

---

## 📂 ファイル構成（主要）
- [index.html](index.html) — アプリのエントリ（ページ構成）
- [script.js](script.js) — アプリのロジック（予約、UI 操作）
- [style.css](style.css) — スタイル
- img/ — ロゴやキャンパス画像（例: `img/logo.png`, `img/takiko.jpg` 等）

---

## 実装上の注意点（今後の改善候補）
- 現在はクライアント側デモのため認証・永続化が簡易実装（実運用ではサーバ側認証とデータベースへ移行推奨）。
- ログイン時に予約を削除する挙動はデモ用であり、実運用では不要／危険。
- 予約衝突判定は分単位で行われますが、UI に分かりやすい表示（カレンダー等）を追加すると良いです。
