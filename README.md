# キャンパスカーシェア予約アプリ（デモ版）

大学のキャンパス間移動を想定した、**カーシェア予約アプリのデモ版**です。  
学籍番号とパスワードでログインし、キャンパス・日時を選択して予約できます。  
利用終了後には自動で**30分のブロック時間**が設定されます。

---

## 🔑 ログイン情報（デモ用）
- 学籍番号：`c000000`  
- パスワード：`000000`

ログイン時、学籍番号はそのまま表示され、  
パスワード入力は「●●●」で隠れる仕様になっています。

---

## 🧭 使い方
1. **ログインページ**  
   - 上記の学籍番号・パスワードでログインします。
2. **キャンパス選択**  
   - 滝子 / 桜山 / 北千種 / 田辺通 から選択。
3. **利用日時選択**  
   - 日付・開始時刻・終了時刻を入力。  
   - 終了時刻の30分後まで自動的に予約不可時間としてブロックされます。
4. **空き確認 → 予約**  
   - 同日同キャンパスの予約が重ならないようにチェックされます。
5. **予約一覧表示**  
   - 当日の予約が一覧で確認できます。
